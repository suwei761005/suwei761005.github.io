劇情簡介

灌籃高手電影版的故事要從一個身高不高卻熱愛籃球的宮城良田說起，小時候時常與哥哥宗太一起打球，

身材高大的宗太用鐵牆鐵壁般的防守讓良田知道，應該要更努力得訓練克服身材上的劣勢，宗太因某次與朋友出海時遇上船難身亡，

良田的母親為此心痛不已，母親不願意觸景傷情而收掉宗太留下來的東西，良田則是一直以哥哥宗太為榜樣，

希望能夠代替宗太守護母親，籃球與宗太的夢想成為宮城良田精神上的唯一寄託，宮城到了湘北高中隨著球隊的訓練，

不管球技及球員每個人都更加團結，湘北高中籃球隊朝著稱霸全國的目標一起前進，此時全國大賽終於開始了，

此次的對手是全日本最強的高中山王工業，面對如此強大的對手，湘北高中是否能克服困境擊敗山王呢?

宗太的夢想是否能由弟弟良田來實現呢?灌籃高手精采的最後一戰湘北與山王工業的對決，

喜歡的各位不妨好好欣賞井上雄彥的大作，灌籃高手電影版THE FIRST SLAM DUNK。

主要人物簡介

宮城宗太

宮城良田的哥哥，時常和良田一起打球訓練，因某次與朋友出海釣魚發生船難身亡，

最大的夢想是打敗山王工業，弟弟良田一直視他為榜樣。

宮城良田

湘北高中控球後衛，與山王的最終決戰被包夾發生多次的進攻失誤，

導致湘北分數落後苦苦追趕，最終依靠自己的速度擺脫了包夾，自此湘北也開始展開反攻。

櫻木花道

湘北高中大前鋒，與山王的決戰中扮演關鍵的角色，不僅多次搶下籃板球，

幾次的灌籃不僅壓制住河田雅史，讓湘北增加更多的反攻氣勢。

三井壽

湘北高中得分後衛，是湘北的得分主力也是分數落後時最主要的進攻人選，

與山王的決戰裡屢屢透過精準的三分球追分，但體力容易下滑影響表現。

流川楓

湘北高中小前鋒，是湘北得分主力之一，個人單打能力、速度、爆發力兼具的球員，

憑藉個人卓越的單打技巧是眾所矚目的潛力新星，缺點是不太傳球。

赤木剛憲

湘北高中中鋒也是現任的籃球隊隊長，在籃下的火鍋封阻、威嚇力、進攻都是湘北最依賴的支柱，

常把球隊勝敗的責任攔在自己身上，此次對上山王的河田雅史不僅籃下被壓制住，

多元進攻手段的河田雅史也讓赤木無法克服內心的恐懼。

澤北榮治

山王工業小前鋒，主要進攻與得分主力之一，個人單打技巧卓越堪稱日本第一，

在與湘北對決之前曾經去寺廟許願，祈求神明可以給他一個寶貴的經驗。

河田雅史

山王工業大前鋒，身高及魁梧的身材在籃下宛如銅牆鐵壁般的防守，

多元的進攻手段讓湘北吃盡苦頭。

深津一成

山王工業控球後衛，聽從教練指示對宮城的包夾防守，

讓宮城在進攻端屢屢發生失誤成為湘北落後的主要原因。

湘北先發五虎未來展望

以下推測若不往籃球領域發展，五個人可能從事的職業類別。

櫻木花道

網紅、演員，櫻木花道本身就充滿自信及內心戲特別多的一個角色，

個人特色可以說是非常鮮明如果當網紅絕對可以做出一番成績，

內心小劇場特多的櫻木對於劇本角色應該會有獨特的見解及詮釋，

如果當演員或許可以拿個最佳新人獎吧。

流川楓

模特兒、作業員，流川楓在劇中除了長得帥、籃球天賦極高，

除了籃球以外好像是個生活白癡或是非常粗心，

應該是可以賣個臉當模特兒，或是當個用勞力賺錢的作業員。

三井壽

公司的主管，自從湘北籃球隊打架事件之後，三井壽歷經滄桑成熟穩重的個性，

在一間公司可以擔任重要職務或主管，是個能夠賦予重任的人。

宮城良田

一人公司(創業)，宮城願意接受身材較矮用自己更多的努力，朝著自己的夢想前進，

這種個性往往人生已經有一個夢想，所以應該會按照自己的想法前進，不太適合去應徵一般的職員。

赤木剛憲

老闆(創業)，赤木是湘北的隊長，是個很有責任感的隊長，

本身以個人為中心去指揮團隊，個性上比較適合創業當老闆，

應該會直接花錢加盟，開餐飲業當老闆。

劇情疑問

《灌籃高手電影版》為什麼這次以宮城良田當主角，而不是櫻木花道呢?

井上雄彥有談到其實在湘北裡面除了宮城其餘的位置都是很有籃球天賦，唯獨宮城受限身材較矮小原本在籃球領域比較沒有優勢，

非常需要依靠後天的努力才能夠有所成就，以這樣的角度去切入一個故事也較能夠貼近一般人，在一般漫畫或動畫中其他腳色背景故事也較為豐富，

此次用宮城當作主角，也希望能夠把這個角色背景故事在描述得更清楚。

《灌籃高手電影版》為什麼電影名稱為THE FIRST SLAM DUNK?

灌籃高手原本的結尾就是全國大賽中湘北與山王的對決，但實際上並沒有在動畫中出現，

因此這次特別把這段做個完整的收尾，電影版的出現對每個灌籃高手迷應該都非常熱血沸騰吧，

而會取名叫做THE FIRST SLAM DUNK，因為裡面劇情有特別調整過，讓第一次接觸灌籃高手的觀眾也不會難以吸收。

《灌籃高手電影版》THE FIRST SLAM DUNK結局是什麼?

湘北在山王對宮城使用了包夾防守讓宮城造成了多次失誤，最終宮城利用果決及速度擺脫了包夾，

讓湘北能扭轉頹勢把落後二十分硬是追趕上來，最後透過櫻木的關鍵投籃讓強敵山王吃上了一場敗仗，

在片尾描述了宮城跟澤北最後都到了美國打球在比賽中相遇。

《灌籃高手電影版》宗太一生的夢想是什麼?

宗太因一次與朋友出海捕魚發生船難而喪生，從小與宗太打球的良田一直視宗太為榜樣，甚至在宗太死後還繼續穿著宗太的球衣，

使用宗太遺留下來的護腕，在一次去海上的洞穴裡良田看到了一本籃球雜誌封面是山王，宗太才說出自己最大的夢想是打敗山王。

《灌籃高手電影版》湘北每個人面對山王所遭遇的困境?

1. 河田雅史不只身材魁梧搶籃板、籃下防守、中距離投籃造成赤木防守極大的壓力。

2. 深津一成及隊友包夾防守造成身材矮小的宮城發生許多進攻上的失誤。

3. 三井壽雖然是湘北得分的主力尤其是三分球屢屢的精準命中，但體力不足讓三井難以幫助湘北緊咬比分。

4. 澤北榮治號稱日本第一，個人單打技巧卓越，不僅守住湘北得分主力流川楓，澤北榮治的進攻實力更成為湘北最大的難題。

《灌籃高手電影版》為什麼宮城的母親要把宗太的東西都收起來?

宮城的母親因失去宗太之後因為太傷心，不想觸景傷情才會把宗太的東西都收起來，

但良田則是一直以哥哥為榜樣，也希望自己能夠代替宗太守護母親。

《灌籃高手電影版》湘北球員經典回憶片段有哪些?

過去的經典片段:

1. 三井與湘北籃球隊打架事件。

2. 赤木在籃球隊訓練中總是以自己為核心聽不進去旁人的建議。

電影版新增的回憶片段:

1. 宗太與良田一起打球。

2. 在失去宗太之后良田與母親家庭關係上出現了變化。

3. 全國大賽前良田仍舊努力的訓練，良田經由彩子的鼓勵與溝通心裡放鬆不少。

4. 在國中時期良田在失去宗太之後曾經跟三井一起打過籃球。

《灌籃高手電影版》彩子與宮城良田是什麼樣的關係?

灌籃高手其實是一部熱血、夢想的一部動畫及電影，所以愛情這一塊並不是主軸，

在劇中也看得出彩子比較多是關心跟鼓勵，就像生命中的一個良師益友一樣，

所以彩子與宮城良田只能說是一個很知心的一個朋友。

觀影心得

在2023年聽到灌籃高手終於要出電影版THE FIRST SLAM DUNK時，只要是民國七十年後的朋友大概跟版主一樣既興奮又期待吧，

小時候重播無數次的灌籃高手終於又回來了，就跟當初灌籃高手手遊一樣非常熱血，記得當時超多youtuber搭上灌籃高手的旋風崛起，

回想當時又辦比賽跟轉播，只能說井上雄彥的灌籃高手真的在很多人心中都佔有相當程度的地位，而這次的劇情以宮城良田的視角，

來描述灌籃高手最終戰湘北對上山王工業，此次針對初次認識灌籃高手的人展現誠意，特別刪減了不少劇情讓第一次看灌籃高手的觀眾更容易了解劇情，

也把宮城良田這個角色的背景故事說得更清楚，全程使用3D動畫讓角色及動作更為立體，以下列出灌籃高手電影版觀影重點。

創新及改變

灌籃高手電影版井上雄彥做了很多的調整，以宮城良田為主角想更貼近凡人的人生，

刪減原作劇情加入更多的原創劇情，使剛開始接觸灌籃高手的觀眾劇情更容易理解，

3D畫風使整體動畫更真實而流暢，一層不變的創作會讓觀眾厭倦，嘗試不一樣的創作跟改變反而讓觀眾有驚喜的感覺，

就像此次的觀影心得我也是做了調整，不單純講劇情更多的是這部電影更全面的觀點。

理想與現實間的拉扯

井上雄彥為什麼不把結局影視化呢?應該是井上雄彥所想的結局跟出版社設定的結局有所出入，

推測是出版社可能不想要在這邊結局，畢竟打造一個熱門的動漫是可遇不可求的，

而井上雄彥想在湘北及山王的對戰畫下完美的句點，才會發生最終的結局並沒有被影視化，

一般劇情應該是會把全國大賽比完，湘北贏得最終的勝利實現了稱霸全國的夢想，

但現實生活中的不完美才是更貼近真實的人生。

中文與日文配音

灌籃高手電影版上映時可以選擇看中文配音或日文配音，我想絕大多數的人應該都是選擇日文配音吧，

但版主是堅持要看中文配音，主要是中文配音才真正活在小時候的回憶裡，

希望再次聽到于正昌老師的宮城良田與于正昇老師的櫻木花道，心裡真的是回憶湧現啊!
